# MovieBox (Моя Кинотека)

A Jetpack Compose prototype implementing the MovieBox brief:

- Popular, search, and favorites tabs with bottom navigation
- Detail-style cards with ratings, notes, and watched state
- Repository + ViewModel abstraction ready for Retrofit/Room integration

## Build

```bash
./gradlew assembleDebug
```

Requires Android SDK 34 and JDK 17.
